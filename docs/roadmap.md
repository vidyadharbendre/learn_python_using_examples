# Roadmap

This roadmap outlines the intended learning path. Update as modules are added.

**Phase 1: Foundations**
- Python basics and core data structures.
- Functions, modules, and packaging.
- Testing fundamentals with pytest.

**Phase 2: OOP and Design**
- Classes, inheritance, composition.
- SOLID principles with exercises.
- Patterns: factory, strategy, adapter.

**Phase 3: Production Practices**
- Typing and mypy.
- Logging and error handling.
- Configuration and dependency management.

**Phase 4: Async and Resilience**
- Asyncio basics and patterns.
- Retries, timeouts, and backoff.
- HTTP clients and rate limiting.

**Phase 5: Capstones**
- Small services or CLIs combining modules.
- Observability and reliability practices.
